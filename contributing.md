# Contributing to Orchid
Orchid is not actively seeking contributors, as the project is still pre-v0. However, if you'd still like to help there are a wide variety of tasks that needs to get done from documentation, website design, and community building.

Things to note if you plan to contribute:

- There is a strict **ban** on the use of any AI for pull requests that deal with the actual `orchid` tool, cryptographic engine, or policy configuration engine.
- AI usage is generally restricted for issues and pull requests, if it is used then it must be noted in the pull request itself.
- Check the issues for tasks to complete, if there's no issue then you can suggest one.
- Keep test coverage high with unit tests and functional tests.

## Ground Rules
We are not in the coding phase yet - still in the dreaded documentation and diagramming phase. Also known as the most important phase of the project. In general, if you'd like to contribute then you'll need to select an issue to work on and coordinate with the project lead.
